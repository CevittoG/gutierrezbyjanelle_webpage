# GutierrezByJanelle - WebSite

A modern, responsive e-commerce platform tailored for independent designers to showcase and sell digital assets. This project focuses on high-performance rendering, SEO optimization, and a seamless checkout experience.

## 🚀 Tech Stack

- **Frontend:** React / Next.js
- **Styling:** Tailwind CSS
- **State Management:** React Context API / TanStack Query
- **Payments:** Stripe API
- **Deployment:** Vercel
